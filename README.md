# NeuroStack CSS Practice

This project is a front-end practice where I recreate a landing page design using HTML and CSS.

The goal of this project is to improve my skills in layout creation, especially using Flexbox to position elements and structure sections.

## What I'm practicing

- HTML page structure
- CSS layout techniques
- Flexbox
- spacing and alignment
- recreating a UI from a design

## Project Goal

The objective is to reproduce the provided design as closely as possible while focusing on understanding how layouts are built.

This project is part of my learning process while studying web development.

## Technologies

- HTML
- CSS
- Flexbox
